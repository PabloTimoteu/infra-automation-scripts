# Infra Automation Scripts

Automação de rotinas de infraestrutura com PowerShell.

## 📌 Objetivo
Centralizar scripts utilizados no ambiente corporativo para otimização, padronização e automação de tarefas administrativas.

## 🔧 Scripts disponíveis

### CacheCleanup-Chrome.ps1
Realiza limpeza de cache de navegadores em perfis de usuários, ajudando a reduzir consumo de disco em servidores.

## 🛠 Tecnologias utilizadas
- PowerShell
- Windows Server
- Active Directory
- KACE SMA
